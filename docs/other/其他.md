# 一些用到的代码

## 正则

```js
参考文章： https://juejin.cn/post/6844903573348696072

用户名： 
字母 + 数字 + @#$%&*_- (8 - 32) 必须包含字母 数字 特殊字符 而且 8 - 32位
/^(?=.*\d)(?=.*[A-Za-z])(?=.*[@#\$%\&*_\-]).{8,32}$/
测试列子
/^(?=.*\d)(?=.*[A-Za-z])(?=.*[@#\$%\&*_\-]).{8,32}$/.test('11111111_o') T
/^(?=.*\d)(?=.*[A-Za-z])(?=.*[@#\$%\&*_\-]).{8,32}$/.test('11111111o')  F 
  
                         
密码：同理 
```
